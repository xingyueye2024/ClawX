# ClawX 项目概览

## 简介

ClawX 是一款基于 Electron + React 构建的跨平台桌面应用，为 [OpenClaw](https://github.com/anthropics/openclaw) AI 代理平台提供图形化操作界面。它将原本需要在命令行中完成的代理编排、网关管理、渠道对接等操作，变成直观的「点点点」桌面体验，让不熟悉 CLI 的用户也能轻松上手 AI 代理的部署与运维。

## 核心定位

| 维度 | 说明 |
|------|------|
| **目标用户** | 不想使用命令行的 AI 代理用户、运维人员 |
| **核心价值** | 安装向导 + 可视化配置，替代命令行和配置文件 |
| **运行方式** | 内嵌 OpenClaw 运行时，开箱即用 |
| **平台支持** | macOS / Windows / Linux |

## 项目特色

### 1. 零配置，开箱即用
- 内嵌 OpenClaw 运行时与 Python 环境管理器（uv），无需用户单独安装网关或核心服务
- 首次启动提供安装向导，引导用户完成语言、AI 提供商密钥、技能包等基础配置
- 自动管理 Gateway 进程的启动、监控和重连

### 2. 多 AI 提供商支持
- 支持 Anthropic（Claude）、OpenAI（GPT）、Google（Gemini）、OpenRouter 等主流提供商
- 支持国内厂商：Moonshot（Kimi）、SiliconFlow
- 支持本地模型部署：Ollama
- 支持自定义提供商配置
- API Key 保存在系统原生钥匙串中，安全可靠

### 3. 丰富的聊天体验
- 流式输出实时展示 AI 回复
- 支持扩展思维（Thinking）模式，可视化 AI 推理过程
- 工具调用状态实时追踪与可视化
- 支持文件附件（图片、文档）
- 多会话管理，可自由创建、切换、删除会话
- Markdown 渲染与代码高亮

### 4. 全渠道消息平台对接
- 支持 WhatsApp、Telegram、Discord、Signal 等 11+ 消息平台
- 支持飞书、企业微信等国内平台
- QR 码扫码登录（WhatsApp）
- Token 认证连接（Telegram、Discord 等）
- 实时连接状态监控

### 5. 技能市场与插件系统
- 集成 ClawHub 技能市场，一键搜索、安装、卸载技能
- 内置核心技能包，开箱可用
- 技能的启用/禁用管理
- 技能配置与版本管理

### 6. 定时任务调度
- Cron 表达式驱动的任务调度系统
- 创建、编辑、启用/禁用定时任务
- 手动触发执行与执行历史追踪
- 下次执行时间自动计算

### 7. 国际化与主题
- 支持三种语言：英语、中文、日语
- 支持浅色/深色/跟随系统三种主题
- 动态切换无需重启

### 8. 自动更新
- 内置自动更新机制
- 支持稳定版 / Beta / 开发版更新频道
- 后台下载，通知安装

## 应用页面一览

| 页面 | 路径 | 功能 |
|------|------|------|
| 聊天 | `/` | AI 对话主界面 |
| 仪表盘 | `/dashboard` | 系统状态总览与快捷操作 |
| 频道 | `/channels` | 消息平台连接管理 |
| 技能 | `/skills` | 技能市场与已安装技能管理 |
| 定时任务 | `/cron` | 计划任务创建与调度 |
| 设置 | `/settings` | 外观、提供商、网关、更新等配置 |
| 初始化向导 | `/setup` | 首次启动引导配置 |

## 项目结构

```
ClawX/
├── electron/          # Electron 主进程（后端）
│   ├── main/          # 应用入口、窗口管理、IPC 处理
│   ├── gateway/       # OpenClaw Gateway 进程管理
│   ├── preload/       # 安全 IPC 桥接脚本
│   └── utils/         # 工具函数（存储、认证、路径、提供商注册）
├── src/               # React 渲染进程（前端）
│   ├── components/    # 可复用 UI 组件
│   ├── pages/         # 应用页面
│   ├── stores/        # Zustand 状态管理
│   ├── lib/           # 前端工具与配置
│   ├── types/         # TypeScript 类型定义
│   ├── i18n/          # 国际化资源
│   └── assets/        # 静态资源与提供商图标
├── resources/         # 打包资源（图标、截图、内置技能）
├── scripts/           # 构建与工具脚本
└── tests/             # 测试套件（Vitest + Playwright）
```
