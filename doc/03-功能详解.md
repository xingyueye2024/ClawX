# ClawX 功能详解

## 1. 初始化向导（Setup Wizard）

首次启动 ClawX 时，用户会进入初始化向导，无需任何技术背景即可完成配置：

**步骤流程：**
1. **语言与地区选择** — 选择界面语言（英语 / 中文 / 日语）
2. **AI 提供商配置** — 输入 API Key 并验证连通性
3. **技能包选择** — 选择需要预装的技能包
4. **配置确认** — 检查所有设置后进入主界面

完成初始化后，应用将自动启动 Gateway 并进入主界面。

---

## 2. 仪表盘（Dashboard）

仪表盘是应用的系统状态总览页面，提供：

- **系统状态概览**
  - Gateway 运行状态（运行中 / 已停止 / 错误）
  - 已连接频道数量
  - 已激活技能数量
  - 系统运行时间（Uptime）

- **快捷操作按钮**
  - 快速跳转到频道管理、技能市场等功能页面

- **已连接频道列表**
  - 显示各频道的连接状态指示器
  - 一览当前活跃的消息平台

- **活跃技能概览**
  - 显示当前启用的技能列表

- **开发者控制台入口**
  - 在开发者模式下可访问 OpenClaw 控制台

---

## 3. AI 聊天界面（Chat）

聊天界面是 ClawX 的核心功能，提供丰富的 AI 对话体验。

### 3.1 消息交互

- **流式输出** — AI 回复以 delta 增量方式实时展示，用户无需等待完整回复
- **Markdown 渲染** — 支持完整的 Markdown 格式，包括代码高亮、表格、列表等
- **消息中断** — 支持在 AI 回复过程中点击中断，停止当前生成

### 3.2 扩展思维模式（Thinking Mode）

- 可通过工具栏切换开启/关闭
- 开启后 AI 会展示推理过程（Thinking Blocks）
- 帮助用户理解 AI 的决策逻辑
- 每个会话可独立配置思维级别

### 3.3 工具调用可视化

- 实时追踪工具/函数调用状态：`运行中 → 已完成 / 错误`
- 展示工具调用的名称与参数
- 可视化工具执行结果

### 3.4 文件附件

- 支持图片上传，自动生成缩略图预览
- 文件通过本地缓存（localStorage）保存，切换会话后仍可查看
- 图片以 Base64 编码发送到 Gateway

### 3.5 会话管理

- **多会话并行** — 可创建多个独立的聊天会话
- **会话切换** — 自由切换不同会话，历史消息完整保留
- **会话删除** — 可删除不需要的会话
- **会话配置** — 每个会话可独立设置思维级别和模型

---

## 4. 频道管理（Channels）

频道管理让用户将 AI 代理连接到各种消息平台，实现多渠道 AI 服务。

### 4.1 支持的频道

| 频道 | 认证方式 | 说明 |
|------|----------|------|
| WhatsApp | QR 码扫描 | 扫码登录 WhatsApp Web |
| Telegram | Bot Token | 通过 BotFather 获取 |
| Discord | Bot Token | 通过 Discord 开发者门户获取 |
| Signal | - | Signal 协议连接 |
| 飞书（Feishu） | App ID/Secret | 飞书开放平台应用 |
| iMessage | - | macOS 原生消息 |
| Matrix | Token | Matrix 协议服务器 |
| LINE | Channel Token | LINE 开发者控制台 |
| MS Teams | App 配置 | Microsoft Teams 集成 |
| Google Chat | 服务账号 | Google Workspace 集成 |
| Mattermost | Token | Mattermost 服务器连接 |

### 4.2 频道操作

- **连接/断开** — 一键连接或断开频道
- **状态监控** — 实时显示各频道连接状态
- **QR 码登录** — WhatsApp 等支持 QR 码扫描认证
- **配置管理** — 管理各频道的认证信息和参数
- **错误展示** — 连接异常时展示错误详情

---

## 5. 技能市场（Skills）

技能系统为 AI 代理提供可扩展的能力插件。

### 5.1 技能浏览与安装

- **市场搜索** — 在 ClawHub 技能市场中按关键字搜索技能
- **一键安装** — 从市场一键安装技能到本地
- **版本管理** — 展示技能版本信息
- **技能详情** — 查看技能元数据（版本、作者、描述、图标）
- **文档查看** — 打开技能 README 文档

### 5.2 技能管理

- **启用/禁用** — 按需开关已安装的技能
- **卸载** — 移除不需要的自定义技能
- **核心技能** — 内置核心技能（不可禁用），保证基础功能
- **配置管理** — 为各技能设置独立参数

---

## 6. 定时任务（Cron）

定时任务功能允许用户创建周期性的自动化任务。

### 6.1 任务创建

- **任务名称与描述** — 为每个任务设定清晰的标识
- **Cron 表达式** — 使用标准 Cron 表达式定义调度计划
- **灵活调度** — 支持分钟、小时、日、周、月等维度的调度

### 6.2 任务管理

- **启用/禁用** — 暂停或恢复任务，无需删除
- **手动触发** — 不必等待调度时间，立即执行任务
- **执行历史** — 追踪任务的历史执行记录
- **下次执行** — 自动计算并展示下次执行时间
- **编辑与删除** — 修改任务参数或彻底移除

---

## 7. 设置（Settings）

### 7.1 外观设置

| 选项 | 可选值 |
|------|--------|
| 主题 | 浅色 / 深色 / 跟随系统 |
| 语言 | 英语 / 中文 / 日语 |

### 7.2 启动行为

- 启动时最小化
- 系统登录时自动启动

### 7.3 Gateway 设置

- 应用启动时自动启动 Gateway
- 端口配置（默认：18789）
- Gateway 重启控制
- 日志查看器（支持打开日志文件）
- 健康状态监控

### 7.4 更新设置

- 更新频道选择：稳定版 / Beta / 开发版
- 自动检查更新
- 自动下载更新
- 手动检查更新
- 当前版本号显示

### 7.5 开发者模式（高级）

开启开发者模式后可访问：
- **OpenClaw 开发者控制台** — Web 管理界面
- **Gateway Token** — 用于手动 WebSocket 连接
- **CLI 命令助手** — 终端使用说明
- **macOS 全局 CLI 安装器** — 安装 `openclaw` 全局命令
